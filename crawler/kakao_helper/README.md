# 카카오톡 자동검색 발송 (PC)

웹 '발송' 버튼을 누르면 → **카카오톡 PC가 켜지고 → 업체명으로 채팅방 자동 검색** → 메시지는 클립보드에 준비됨.
그 다음 **뜬 채팅방 클릭 → Ctrl+V → Enter** 로 직접 전송. (자동 대량발송이 아니라 사람이 보냄 = 안전)

## 설치 (한 번만)
1. **카카오톡 PC**를 실행해 둔다.
2. 명령 프롬프트에서:
   ```
   cd C:\Users\ddmkt\DDMKT_ERP\crawler\kakao_helper
   python setup_kakao.py
   ```
   - `ddmkt-kakao://` 프로토콜이 등록되고(관리자 권한 불필요),
   - 검색창 위치 보정: 안내가 나오면 **카톡 '검색창'에 마우스 커서를 올려둔 채** 6초 기다리면 저장됨.

## 사용
- 웹(크롬/엣지)에서 **발송** 클릭 → 처음 한 번 "DDMKT Kakao 열기" 허용(다시 안 물어보게 체크).
- 카톡이 켜지고 업체명이 검색됨 → **채팅방 클릭 → Ctrl+V → Enter**.

## 검색 위치가 안 맞을 때
- 카톡 창 크기/위치를 바꿨으면 `python setup_kakao.py` 다시 실행해 보정.
- 보정 정보는 `kakao_config.json` 에 저장됨(창 비율 기준이라 위치 옮겨도 어느 정도 따라감).

## 참고
- 모바일은 이 헬퍼가 필요 없음 — 폰에서 발송하면 카카오 SDK로 바로 카톡 채팅방 선택창이 뜸.
- 헬퍼 미설치 PC에서도 발송 누르면 **메시지가 클립보드에 복사**되므로 직접 검색·붙여넣기 가능(그냥 자동검색만 안 될 뿐).
